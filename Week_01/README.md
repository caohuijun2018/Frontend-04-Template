3.异步编程
 1) callback  函数回调
function go() {
        green()
        setTimeout(function () {
            yellow()
            setTimeout(function () {
                red()
                setTimeout(function () {
                    go()
                }, 2000);
            }, 3000);
        }, 5000);
    }

 2）promise  链式表达式的形式
function sleep(t) {  
            return new Promise((resolve,reject) => {
                setTimeout(resolve,t);
            })
        }
        function go() {
            green()
            sleep(5000).then(() => {
                yellow();
                return sleep(3000);
            }).then(() => {
                    red();
                    return sleep(2000);
                }).then(go);
        }

 3）async/await
在promise的基础上进行语法上的改进，优点在与可以更换处理函数
async function go() {
            while (true){
                green();
                await sleep(5000);
                yellow();
                await sleep(3000);
                red();
                await sleep(2000);
            }
        }
